==== İlk Kez Kullanım ====

unfoldingWord uygulaması bir cihazda ilk kez açıldığında dil seçim sayfası açılır. Görüntülemek istediğiniz dile dokunun. Öyküler o dilde açılacak. Bu uygulamanın değiştirilebilecek başka ayarları yoktur.

==== Checking Levels ====

Below the list of languages, there is an explanation of the three-level checking system. Each level is described in detail.

==== Language Information ====

To view the checking status of each language translation, tap the **  // i  //  **    icon to the right of the language. The screen will open to show the translators for that language, the checking level, version and publish date.
