==== tK’yi Nasıl Kurabilirim? ====

{{:en:ta:tech:setting_up_translationkeyboard-android_v2.mp4|Setting Up the translationKeyboard}}

Şu üç dakikalık videoyu izleyin ya da şu talimata uyun: dilinizdeki karakterlerle yazabilmeniz için translationKeyboard’u kurun. Programı açmak için tK simgesine dokunun. Kurulum Talimatları penceresi açılacak. Ekrandaki bilgileri hareket ettirmek için parmağınızla dokunup yukarıya çekin.

==== 1. Adım: translationKeyboard’u sisteme ekleyin ====

  * «Klavyeyi etkinleştirin» düğmesine dokunun.
  * Bütün mevcut olan klavyelerin bulunduğu pencere açılacak.
  * Listede translationKeyboard bulun.
  * translationKeyboard adından sağa kayan on-off düğmesini arayın.
  * Düğmeye dokunup parmağınızla kaydırın. Düğme yeşil renge dönecek.
  * İkaz penceresi açılacak.
  * Klavyenin çalışmasına izin vermek için «OK»e dokunun.
  * Android menüsündeki kurulum penceresine geri dönmek için geri oka dokunun.

==== Action 2: Set the input method ====

   * Tap the 'set input method' button.
   * A window opens. Tap the circle by 'translationKeyboard'.
   * The window closes.

==== Action 3: Select the input languages ====

   * Tap on the 'Input Language' button.
   * The Input languages list shows.
   * Use a tap-swipe-up action to see all the languages on the list.
   * Look for the target language keyboard.
   * Select by tapping on the box to the right of the language.
   * Use the back arrow on the Android menu to go back to the setup window.
